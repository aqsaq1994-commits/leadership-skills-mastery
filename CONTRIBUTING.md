cat << 'EOF'
# Contributing to Leadership Skills Mastery

Thanks for your interest in improving this curriculum! We welcome contributions from instructional designers, developers, educators, and anyone passionate about leadership development.

---

## 🤝 How to Contribute

### **Option 1: Report an Issue**
Found a typo, bug, or have a suggestion?

1. Go to the [Issues](https://github.com/aqsaq1994-commits/leadership-skills-mastery/issues) tab
2. Click **"New Issue"**
3. Describe what you found
4. Click **"Submit new issue"**

### **Option 2: Submit Changes (Pull Request)**

1. **Fork the repository** (click "Fork" button, top-right)
2. **Clone your fork to your computer:**
   ```bash
   git clone https://github.com/[YOUR-USERNAME]/leadership-skills-mastery.git
   cd leadership-skills-mastery
   ```
3. **Create a feature branch:**
   ```bash
   git checkout -b improve/your-idea
   ```
4. **Make your changes** (add lessons, improve components, fix typos, etc.)
5. **Commit with a clear message:**
   ```bash
   git commit -m "Add [what you did]. Fixes #[issue number if applicable]"
   ```
6. **Push to your fork:**
   ```bash
   git push origin improve/your-idea
   ```
7. **Open a Pull Request** on the main repo with a description of changes

---

## 💡 Contribution Ideas

We'd love help with:

- **Additional lessons** for any module
- **New interactive components** (React-based learning activities)
- **Case studies** demonstrating frameworks in action
- **Translations** (Spanish, Mandarin, Arabic, etc.)
- **Accessibility improvements** (WCAG 2.1 AA compliance)
- **Feedback & assessment tools**
- **Visual diagrams** for leadership frameworks
- **Research citations** and bibliography expansion
- **Facilitator guide** for group learning
- **Typo fixes** and clarifications

---

## 📋 Code Standards

Please follow these guidelines:

### **Markdown Lessons**
- Clear headings (H1, H2, H3)
- Bullet points for lists
- Include learning objectives at the top
- End with reflection questions or action steps

### **React Components**
- Functional components using hooks
- Clear comments explaining logic
- Accessible (keyboard navigable, color contrast ≥4.5:1)
- Responsive design (works on mobile + desktop)
- No external dependencies (use pure React + CSS)

### **CSS**
- Use CSS variables (defined in `:root`)
- Semantic class names
- Mobile-first responsive design
- WCAG 2.1 AA color contrast
- Include dark mode support

### **General**
- Clear, professional tone
- Proper grammar and spelling
- Test before submitting

---

## 🔍 Review Process

1. **Submit** → We review your contribution
2. **Feedback** → We may suggest improvements
3. **Iterate** → Make updates if needed
4. **Merge** → Your changes become part of the course!

---

## ❓ Questions?

- Open a [Discussion](https://github.com/aqsaq1994-commits/leadership-skills-mastery/discussions)
- Review [COURSE_ARCHITECTURE.md](COURSE_ARCHITECTURE.md) for design philosophy
- Check existing issues/PRs to see what's in progress

---

## 🙏 Thank You

Every contribution—big or small—helps make leadership development better for everyone. We appreciate you!

---

**Let's grow leaders together.** 🚀
EOF
